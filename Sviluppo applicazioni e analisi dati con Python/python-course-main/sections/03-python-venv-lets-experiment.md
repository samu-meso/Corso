# Python Installation and Fundamental Tools: Let's Experiment!

## Exercise 1: 

Prepare a virtual environment for the «***Software Development with Python***» course. 

The environment must be placed in the working directory of your projects (e.g., on Windows: `C:\Users\username\workspace\courses\software-with-python`; on Linux: `/home/username/workspace/courses/software-with-python`), within the folder `/software-with-python/theory`, and must meet the following requirements:

1. The name of the environment must be `venv`;
2. The environment must be complete with the file `requirements.txt` generated by the command line;
3. The environment must contain a hand-written `README.md` file containing the version of Python you are using.

--

## Exercise 2: 

For each exercise proposed hereafter, create a folder at the path `/software-with-python/exercises` named `exercise_name`, and install the relative virtual environment inside it. 

Each environment must meet the following requirements:
1. The name of the environment must be `venv`;
2. The environment must be complete with the command-line generated `requirements.txt` file;
3. The environment must contain a hand-written `README.md` file containing the Python version you are using.
